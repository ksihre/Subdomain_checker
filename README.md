#Subdomain Checker

# Overview
A Python-based tool that identifies valid subdomains for a given domain. This project demonstrates basic reconnaissance techniques used in cybersecurity for discovering publicly accessible subdomains.

#Features
- Accepts a target domain as input
- Checks common subdomains
- Identifies active subdomains
- Simple and easy to use

#Technologies Used
- Python
- Requests library
- Networking concepts

#How It Works
The tool sends HTTP requests to potential subdomains of a target domain. If a valid response is received, the subdomain is considered active.

# How to Run

1. Make sure Python is installed  
2. Install required library:
    pip install requests

3. Run the script:
    python subdomain_checker.py

4. Enter the target domain when prompted  

#Learning Outcomes
- Understanding of subdomain enumeration  
- Basics of reconnaissance techniques  
- Working with HTTP requests in Python
